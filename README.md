## Capabilities
1. BruteForce Account.
2. Get Information of any Account.
3. UserChecker.
4. User Grabber.

## [+] Installation and Usage
* $ apt update
* $ apt upgrade
* $ pkg update
* $ pkg upgrade
* $ pkg install git
* $ pkg install python
* $ pkg install python2
* $ git clone https://github.com/r0cketp0wer/InstaBox
* $ cd InstaBox

Now from here choose any module you want to use e.g  $ python BruteForcer.py
